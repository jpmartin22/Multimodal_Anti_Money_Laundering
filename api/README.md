# API Directory

Store FastAPI application code and REST API endpoints here.

## Structure

- **`main.py`** — FastAPI application entry point and route definitions
- **`schemas.py`** — Pydantic models for request and response validation
- **Other modules** — API utilities, middleware, dependencies

## Usage

```bash
# Run development server
uvicorn multimodal_anti_money_laundering.api.main:app --reload
```
